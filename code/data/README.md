# Data Directory

This directory contains the cleaned panel data used in the empirical analysis.

## File Description

- `panel_40countries.csv`: Cleaned panel data for 40 countries, 1990-2024

## Data Sources

- WID 2026 (World Inequality Database): wealth share data (top10, bottom50, top1)
- WGI (Worldwide Governance Indicators): Rule of Law, Regulatory Quality
- Edelman Trust Barometer 2025: social trust index
- Penn World Table 10.0: macroeconomic variables (GFCF, GDP, GDP deflator)
- Chetty et al. (2014): intergenerational mobility index

## Usage Notes

All data users must comply with the citation requirements of the original sources:

- WID: https://wid.world
- WGI: Kaufmann, D., Kraay, A., & Mastruzzi, M. (2010)
- Edelman: Edelman Trust Barometer 2025
- PWT: Feenstra, R. C., Inklaar, R., & Timmer, M. P. (2015)
- Chetty: Chetty, R., Hendren, N., Kline, P., & Saez, E. (2014)
